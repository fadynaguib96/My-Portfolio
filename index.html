Word Guess Game

This is a basic hangman game, in a random soccer team is picked at random and displayed as empty lines. The player would then have seven tries to try to guess the correct letters in the word. The game automatically resets after the player either wins or loses. Go ahead and try it out!

Crystal Collector Game

This is a crystal collector game. Four crystals are displayed, each of them with a random set score between 1-12. The objective of the game is to try to match the score assigned by the computer excatly, no more and no less. Go ahead and try it out!

Giftastic

This is a fun giphy website with a hip-hop theme. Buttons of famous hip-hop artist are displayed on the main page. Once clicked, 10 up-to-date giphy are display on the poge used the Giphy API. The user can also add his/her own artists to check out their latest gifs. Go ahead and try it out!

Trivia Game

This is a Russia 2018 World Cup game. The player will have to answer 10 questions about the world cup, each question within a 20 second period. If the time runs out, the player doesn't record an answered question and the next question is displayed. Go ahead and try it out!

Train Scheduler

This is a handy train scheduler website that allows the admin to display the latest trains, destination, next arrival times, and minutes to arrival. This website uses firebase to store all train information along with the moment.js library to display the next arrival and minutes to arrival sections of the website. Go ahead and try it out!
